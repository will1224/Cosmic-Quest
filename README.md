# Cosmic-Quest

Cosmic Crusaders 🌌

Cosmic Crusaders is a fun, interactive trivia game designed to teach children aged 5-10 to teach them about the solar system. Players read short passages about celestial bodies and then answer trivia questions to test their knowledge!

**Features**

🌟 Educational Content: Teach kids about planets, moons, stars, and more in an engaging way.
🛸 Interactive Gameplay: Read passages and answer trivia questions to earn points.
🚀 Child-Friendly Design: Vibrant visuals and intuitive controls designed for young learners.
💫 Progress Tracking: Keep track of your child's learning journey through levels.


**How to Play**

1. Start the Game: Launch the game from your computer.
2. Read the Passage: Learn about celestial bodies through short, engaging texts.
3. Answer Questions: Answer multiple-choice trivia questions based on the passage.
4. Score Points: Earn points for correct answers and unlock new content as you progress!
5. Explore the Cosmos: Discover fun facts and expand your knowledge of space.


Installation

Prerequisites
Node.js
Git
Steps
Clone the repository:
git clone https://github.com/your-username/cosmic-crusaders.git
Navigate to the project directory:
cd cosmic-crusaders
Install dependencies:
npm install
Start the development server:
npm start
Open the game in your browser at http://localhost:3000.
Tech Stack

Frontend: HTML5, CSS3, JavaScript, React
Backend: Node.js, Express.js
Database: MongoDB (optional for saving user progress)
Design Tools: Figma for prototyping
